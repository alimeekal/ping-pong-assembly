# Ping Pong — x86 16-bit DOS Assembly

A two-player Ping Pong game written entirely in x86 16-bit Assembly, built to run as a `.com` file on DOSBox.

---

## Features

- Two-player local multiplayer on the same keyboard
- Smooth ball movement with wall and paddle reflection
- Score tracking for both players
- Configurable winning score (5, 7, or 9 points)
- Pause and resume mid-game
- Welcome screen with instructions
- Win/exit screen at the end

---

## Requirements

- [NASM](https://www.nasm.us/) — Netwide Assembler
- [DOSBox](https://www.dosbox.com/) — DOS emulator

---

## Build & Run

**1. Open DOSBox and mount your folder:**
```
mount e C:\path\to\folder
e:
```

**2. Assemble the source file:**
```bash
nasm pingpong.asm -o pingpong.com
```

**3. Run the game:**
```
pingpong.com
```

## Controls

| Action | Player 1 (Left) | Player 2 (Right) |
|---|---|---|
| Move Up | `W` | `↑` Arrow |
| Move Down | `S` | `↓` Arrow |

| Key | Action |
|---|---|
| `P` | Pause / Resume |
| `ESC` | Exit game |

## How It Works

### Language & Environment
Written in x86 16-bit real-mode Assembly. Runs as a flat `.com` binary under DOS.

### Ball Movement
The ball moves each frame by adding a direction offset to its video buffer position. It reflects off the top and bottom walls and off the paddles.

### Scoring
When the ball crosses the left or right boundary, the opposing player scores. The score display is updated directly in the video buffer. First to reach the chosen winning score wins.
